Lesson 2: Minimal Program Framework

What is a Framework?

A framework is like a house where you put furniture. This framework is a simple Python program where you can put OpenCV code to see how it works.

Important Commands

import cv2

This brings OpenCV tools into your program. Like opening a toolbox.

cv2.imread('image.jpg')

Reads an image from a file. Your drone uses this to see pictures it takes.

cv2.imshow('Window', image)

Shows the image in a window. Like displaying a photo on screen.

cv2.waitKey(0)

Waits for you to press a key. Without this, the window closes too fast!

Complete Framework Code

# Drone camera starter code
import cv2

# 1. Read image from drone camera
# Replace 'drone_photo.jpg' with your image
image = cv2.imread('drone_photo.jpg')

# 2. Check if image loaded correctly
if image is None:
    print("Error: Could not read image")
    print("Make sure 'drone_photo.jpg' exists!")
else:
    # 3. Show the image
    cv2.imshow('Drone View', image)
    
    # 4. Wait for key press
    print("Press any key in the window to close...")
    cv2.waitKey(0)
    
    # 5. Close all windows
    cv2.destroyAllWindows()

print("Program finished!")
